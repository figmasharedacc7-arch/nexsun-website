## TIPS

1. **Generate 4 variations per prompt** then iterate on the strongest. Most platforms include this by default.
2. **Add `--seed 1234`** in Midjourney to lock a composition then iterate on style.
3. **For the live hero**, you'll likely want **option 1, 2, or 6**. Options 3, 4, 5 work better for sub-sections.
4. **Aspect ratio**: 16:9 for hero composition, 1:1 for standalone sphere assets.
5. **Resolution**: Use Midjourney's `--upscale` or DALL-E's HD mode for production assets.
6. **Save the rendered images** at `nexsun/experiments/concepts/assets/` so we can swap them into the HTML experiments.
