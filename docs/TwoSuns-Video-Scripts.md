# TwoSuns.ai — Website Video Scripts

**Purpose:** Replace the six legacy videos currently on the site (1 homepage brand film + 5 module videos on the Modules page) with new, on-brand TwoSuns videos. This document is the **content/script stage**, for review and supervisor sign-off, before any production begins.

**Status:** Draft v1 for review · Owner: Raihaan · Date: 2026-06-05

---

## How to read this document

Each video has a scene-by-scene breakdown with four columns:

- **Time** — running timecode within the clip
- **Visual** — what is on screen / motion direction
- **On-screen text** — captions/words shown (these carry the message even with sound off)
- **Voiceover (VO)** — optional narration if we produce a narrated version

The site's videos currently autoplay **muted and looping**, so every script is written to work **silent** (on-screen text carries it). The **VO** lines are included so we can also cut a narrated version for sales decks, LinkedIn, and YouTube from the same script. Pick one or both per video at production time.

## Positioning assumption (please confirm with supervisors)

These scripts use the **industrial positioning** direction: TwoSuns as *the orchestration layer for complex industries*, with named verticals **Cement, Mining, Manufacturing, Transportation & Logistics, Clean Tech, and Industrial Operations**. If leadership prefers the broader "complex organizations" framing instead, only the vertical names and a few lines change; the structure stays the same.

## Brand & motion style guide (applies to all six)

- **The two suns are the hero motif.** Gold sun = intelligence and insight. Orange sun = execution and action. Across the films they move from scattered/apart to aligned/orchestrated. This is the visual thesis of the whole brand: two forces, one coordinated system.
- **Palette:** Navy `#14304F`, Gold `#D9B341 / #F6B707`, Orange `#F4920B / #FF6600`, Cream `#FFFBF0`. Dark navy for depth, warm golds/oranges for the light and energy.
- **Tone:** Institutional, precise, quietly confident. Enterprise, not consumer. No hype, no stock-tech clichés (no glowing brains, no binary rain).
- **Motion principle:** Calm and deliberate. Fragmentation resolving into order. One meaningful movement per scene, not constant churn.
- **Type:** Bold sans for statements, clean UI type for product moments.
- **Specs:** 1920×1080 (16:9) master; module clips also exported at the page's landscape crop. 24–30 fps. H.264/MP4 for web (muted loop, 10–22s) plus an optional captioned/narrated cut (20–45s). Keep web loops under ~6 MB, self-hosted (not Framer CDN).
- **End card (all videos):** the TwoSuns wordmark with the two-sun mark, tagline **"Orchestrating Execution."**
